# Sanity Content Guide - Matching site-content.json

This guide shows exactly what content to enter in Sanity Studio to match your `site-content.json` file.

## Stats Section (Must have exactly 3 stats)

Add these 3 stats in order:

1. **First Stat:**
   - Label: `Subscribers`
   - Value: `500+`

2. **Second Stat:**
   - Label: `Reports`
   - Value: `200+`

3. **Third Stat:**
   - Label: `Prospects`
   - Value: `50+`

⚠️ **Important**: The schema now requires exactly 3 stats. You'll see an error if you try to add more or less than 3.
